# To Run 
## Starter
Assumes you have Conda installed.
### Create Conda Env
`conda create -n gan`
### Activate Env
`conda activate gan`
### Install PIP 
`conda install pip`
### Install Requirements
`pip install -r requirements.txt`

# To Use
Iterate through each line of the selected notebook to achieve a working GAN. Adjust EPOCH and other Setup as needed.
